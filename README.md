# Mi_aprendizaje_JavaScript
## ¡Bienvenido/a a mi Cuaderno de Programación en JavaScript!

Este repositorio es una colección personal de apuntes, ejemplos de código y resúmenes de conceptos clave que he ido documentando a lo largo de mi viaje de aprendizaje en JavaScript. Es mi forma de consolidar conocimientos, practicar la documentación y tener una referencia rápida de los temas más importantes.
